EN
Java program that checks whether a directory entered by the user is a palindrome. Palindrome is called sentences, 
words and numbers that are the same backwards.

In the program; The user enters an array. The program reverses it and creates a new string. 
Finally, we compare it to the input string.

Ex= noon, 121 , refer , racecar , toot ...
